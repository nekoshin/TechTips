Git for Windows(msysGit)

今は名前がmsysGitからGit for Windowsに変わっているらしく、

Git for Windows(msysGit)
http://msysgit.github.io/

からダウンロードできる。
環境変数の適用をGit Bashだけにしてたけれども、cmd.exeやnyaosから使用したかったので、 結局手動でpath追加した気がする。

Initial configuration

ユーザ名とメールアドレスの設定

$ git config --global user.email "you@example.com"
$ git config --global user.name "Your Name"
$ git config --global color.ui auto

--globalなので、そのシステム上（すべてのrepository）で使用されるものとなる。 異なる識別情報を設定したいときは、--globalなしで設定する。
最後のは見やすくするための設定。

SSH

先にユーザディレクトリに.sshフォルダを作成しておく。
windowsであれば、C:\users\ユーザ名\.sshのような感じで。

$ ssh-keygen -t rsa -C "username@example.com"
Generating public/private rsa key pair.
Enter file in which to save the key (//.ssh/id_rsa): C:/users/username/.ssh
sa
Enter passphrase (empty for no passphrase):
Enter same passphrase again:
Your identification has been saved in C:/users/username/.ssh/id_rsa.
Your public key has been saved in C:/users/username/.ssh/id_rsa.pub.
The key fingerprint is:
フィンガープリントが表示される username@example.com

これでid_rsa,id_rsa.pubが生成される
公開鍵を表示したいときは 

$ cat id_rsa.pub

nyaos使ってると、何気にcatと打っても表示してくれるから地味に助かる。
テキストエディタなどで表示するのは、うまくいかない。
