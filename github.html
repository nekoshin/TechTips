Bitbucket

First commit

$ cd C:/repository\reponame\	
$ git init
Reinitialized existing Git repository in C:/repository/reponame/.git/

$ git remote add origin https://xxxx@bitbucket.org/xxxx/reponame.git
fatal: remote origin already exists. ← すでに作られている場合

$ echo test >> README.md

$ git add README.md
commitをする際にemailとユーザ名が設定されていないときは下記が表示される。 
*** Please tell me who you are.

Run

  git config --global user.email "you@example.com"
  git config --global user.name "Your Name"

to set your account's default identity.
Omit --global to set the identity only in this repository.
--globalなので、そのシステム上（すべてのrepository）で使用されるものとなる。 異なる識別情報を設定したいときは、--globalなしで設定する。
